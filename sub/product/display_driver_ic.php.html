DDI

Source Driver IC

Source driver LSI for AMOLED panel

- 768/640ch driving output channels
- EPI interface: Embedded clock Point to point interface.
  • : Lock signal cascading and feedback to T-Con
  • : Embedded Control Data
  • : Maximum Bandwidth: 1.61Gbps/pair
  • : Embedded termination resister
- 1024 gray scale per output
- Adoption of LbR port for easy OLED wiring OLED compensation function
- 192/160 compensation channels
- Embedded 10bit ADC with 11MHz sampling rate
- 100Mbps bLVDS Tx to transfer the 10bit data to MCU Power supply voltage
- Analog HV: 12 to 18V
- Low voltage for logic, ADC, & EPI: 1.62 to 1.98V Only COF (Chip On Film)

Gate Driver IC

Target Mode : AMOLED APPLICATION

Operation frequency: Maximum 400kHz (2.5us line time)

Output channels: 432/360 outputs

Gate line voltage: MAX. 40V

- Positive output voltage : VGH = 10 ~ 35V
- Negative output voltage : VGL-VSS = -5 ~ -12V
Supply voltage for Logic Signal : 2.7V ~ 3.6V

Function

- Internal POR (Power-on Reset) Circuit Adopted
- Bi-directional Shift Registers Adopted
- Output Masking Function Adopted
- Symmetrical Input Pin Allocation for Gate PCB_less Application
- Shift by three or four stages and many drive algorithms for AMOLED TV
PKG : COF(Chip On Film) OR COG(Chip On Glass)
